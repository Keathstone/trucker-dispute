# Trucker Dispute — Backend Server

## One-Click Deploy (Render)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Keathstone/trucker-dispute-server)

Then add env var: `ANTHROPIC_API_KEY`

## One-Click Deploy (Railway)

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/https%3A%2F%2Fgithub.com%2FKeathstone%2Ftrucker-dispute-server)

Then add env var: `ANTHROPIC_API_KEY`

---

## Manual Deploy (anywhere)

```bash
git clone https://github.com/Keathstone/trucker-dispute-server
cd trucker-dispute-server
cp .env.example .env
# edit .env with your ANTHROPIC_API_KEY
npm install
npm start
```
